HTML

HMTL tags

  • Standard HTML Tags

Tag Fonctionnality
!DOCTYPE html Declare HTML document
html Indicates beginning and end of html code
head Encapsulates metadata (title, link to stylesheet,...)
body Encapsulates content to be displayed in the browser
div Division of content
img Embeds image in webpage. Self-closing tag
video, audio Embeds media on webpage
p paragraph

  • Semantic HTML tags

    Semantic HTML allows for better code readability and maintenance, as well as better parsing from search engines.

    They are tags designed to cut the content into easy-to-identify pieces, and should be used in the body part of the html code.

Tag Usage
header Should contain the head of the webpage (title, main nav bar..)
nav Used to identify a navigation menu
article Used to identify an article (a part of a webpage that has meaning on its own, independant of the rest of the page.)
section Used to section the webpage in different sections
aside Used for content that should be presented next to the main content (highlight quote, sidebar,...)
footer Used for footer of article or webpage (contact info, signature,...)

Tables

Tables need to be declared using the <table> and </table> tags.

Tag Usage
thead Identifies head of the table
tbody Identifies actual content of the table
tr Creates new row
td Create new cell and fills it with data
tfoot Identifies foot of the table

CSS

Generalities

CSS, short for Cascading Style Sheet, is a language used to change the appearence and disposition of content on a webpage.

While inline CSS Styling is possible directly in an html document, the preferred methode is to link to a dedicated stylesheet.

The syntax in this dedicated stylesheet is : selector { attribute : value of attribute}

Example : p {font-familiy: Helvetica} will change all paragraphs fonts to Helvetica.

CSS selectors

HTML content can be targeted for CSS styling using selectors. They are, by increasing order of specificity :

Selector Syntax targeted HTML
Type selector h1{}, p{}, body{},... General HTML tags : <p>,<body>,...
Class selector .example{}, .main{},... Attribute within html tag : <div class="main">
Id selector #id{}, #menu{},... Id attribute within html tag : <div id="menu">

Selectors can be combined to :

  • target more specifically : .main section will target every main element that is child to a section element
  • target multiple elements: h1,p,footer will apply the same style to the p, h1 and footer elements
